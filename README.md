# SP-III — Git Learning Project (HTML/CSS/JS)

This repository contains a simple, well-organized **HTML/CSS/JavaScript** project created for a Git Version Control assignment.  
The web page is a Bangla-friendly **Git command guide** with navigation, search, and animated UI.

## Project structure

- `index.html` — main page
- `css/style.css` — styles
- `js/script.js` — interactivity (scroll, search, highlights)
- `CHANGELOG.md` — release notes
- `.gitignore` — ignored files

## How to run

- Open `index.html` in a browser, or
- Use VS Code “Live Server” extension (recommended).

## Assignment checklist (Git features)

This repo will demonstrate:
- 10+ meaningful commits with clear messages
- 3+ branches (example: `feature/ui`, `feature/docs`, `feature/refactor`)
- Proper merges into the main branch, including at least one **merge commit**
- An intentional **merge conflict** and manual resolution
- Undo operations using `git revert` and `git restore` / `git reset`
- Recovery using `git reflog`
- Release tag `v1.0.0`
